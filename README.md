### 🔧 사용 언어
![](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)

### 💼 사용 프레임워크
 - PyQt5

### ⚙️ 사용 api
- Cerebras Cloud (Llama-3.3-70B)

---

### 구조
- **Editor Window**
    - 에디터를 위한 메인 윈도우를 구성합니다. UI처리 및 상태 관리를 담당합니다.
- **File Manager**
    - 파일 관리를 담당합니다.
- **Terminal Manager**
    - 에디터에 탑재될 Terminal의 기능 관리를 담당합니다.
- **Agent Manager**
    - 파일 에이전트 관리를 담당합니다.
- **Model Manager**
    - LLM 모델을 관리합니다.
